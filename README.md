# ChatGPT Chat Hider (Chrome Extension)

ChatGPT の会話で不要な質問・回答やアクションバー（コピー/再生成など）を簡単に非表示にできる拡張です。

## 使い方

1. Chrome の拡張機能管理画面を開きます。
2. 「デベロッパーモード」を有効にして、このフォルダを「パッケージ化されていない拡張機能を読み込む」で読み込みます。
3. ChatGPT を開くと、右下にコントロールが表示されます。

## 機能

- 各質問/回答に「Hide」ボタンを追加して個別に非表示（チャット移動・再読み込みでも維持）。
- 「Hide all Q/A」ボタンで一括で質問・回答を非表示。
- 「Show all Q/A」で一括表示。
- 「Hide action bars」で、コピー/再生成などのアクションバーをまとめて非表示。
